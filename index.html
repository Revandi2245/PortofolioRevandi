Tentang Saya

Saya seorang mahasiswa/peserta PKL yang ingin mencari pengalaman di industri terkait pengembangan software dan analisis data. Selama praktik kerja lapangan, saya ingin terlibat dalam berbagai tugas pengembangan perangkat lunak termasuk menganalisis tren data untuk rekomendasi bisnis, mengumpulkan dan mengolah data dari berbagai sumber, serta menganalisis kebutuhan sistem untuk pengembangan software. Saya juga berkeinginan untuk berkontribusi dalam pengembangan website dengan membangun antarmuka pengguna (frontend) dan logika sistem (backend). Saya berkomitmen untuk mengembangkan hard skill seperti penguasaan tools analisis data (Python, SQL, Excel, Talend), kemampuan ETL dan visualisasi data, serta pengembangan frontend-backend. Di sisi soft skill, saya ingin meningkatkan kemampuan analisis, ketelitian, komunikasi, kolaborasi tim, dan pemecahan masalah. Pemahaman tentang SDLC (Software Development Life Cycle) juga menjadi fokus pengembangan saya selama PKL ini. Melalui pengalaman PKL, saya berharap dapat menerapkan pengetahuan akademik dalam proyek nyata sekaligus belajar dari praktisi industri untuk mempersiapkan karir di bidang pengembangan software dan analisis data.
